# QuickRider
Rider app for **QuickFoodRide**.

## Features:
* **New Ride notification**
* **Ride details**
* **Customer info**
* **Accepting ride**
* **Ride canellation**
* **Push notification**
* **Ride history**
* **Rating customers**
